# Saibalaji Neeli — Portfolio

Personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just one file.

🔗 **[Live Site](https://SaibalajiN.github.io/saibalaji-neeli)**

---

## Features

- Animated particle constellation background that reacts to cursor movement
- Text scramble effect on hero name
- Typing animation cycling through roles
- 3D card tilt effect on project and skill cards
- Magnetic button effect
- Cursor trail animation
- Scroll progress bar
- Animated counters on KPI stats
- Skill radar chart and animated progress bars
- Dark / light mode toggle
- Mobile responsive with hamburger menu
- Project detail modals
- Scrolling tech stack marquee
- Smooth scroll fade-in animations throughout
- Contact form with mailto integration

## Sections

- Hero with availability badge
- About Me
- Projects with live demos and detail modals
- Skills with radar chart and progress bars
- Work Experience timeline
- Education
- Research / Publications
- Certifications
- Testimonials
- Contact

## Tech Stack

Built with zero dependencies — pure HTML, CSS, and vanilla JavaScript.
